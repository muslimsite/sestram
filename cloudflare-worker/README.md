# Отзывы → Telegram-группа (через Cloudflare Worker)

Форма «Оставить отзыв» на сайте отправляет отзыв не напрямую в Telegram, а на
маленький бесплатный сервис **Cloudflare Worker**. Worker хранит токен бота
**скрытно** (его не видно в коде сайта) и пересылает отзыв в вашу Telegram-группу.

Отзывы приходят **только вам в группу** — на сайте автоматически они не публикуются
(модерация). При желании понравившийся отзыв вы добавляете на сайт вручную.

Настройка делается один раз и занимает ~10 минут.

---

## Шаг 1. Создать бота в Telegram

1. Откройте в Telegram **@BotFather**.
2. Отправьте `/newbot`, придумайте имя и username бота.
3. BotFather пришлёт **токен** вида `123456789:AAH...` — скопируйте и сохраните его.

## Шаг 2. Добавить бота в группу

1. Создайте группу (или используйте существующую), куда будут приходить отзывы.
2. Добавьте в неё своего бота как участника.
3. Желательно сделать бота **администратором** группы (чтобы он точно мог писать).

## Шаг 3. Узнать ID группы (`CHAT_ID`)

Самый простой способ:

1. Добавьте в группу бота **@getidsbot** (или **@username_to_id_bot**).
2. Он покажет ID группы — число вида `-1001234567890` (для групп оно **отрицательное**).
3. Скопируйте это число. Бота-помощника после можно удалить из группы.

> Альтернатива без сторонних ботов: напишите любое сообщение в группе, затем откройте
> в браузере `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates` и найдите в ответе
> `"chat":{"id":-100...}`.

## Шаг 4. Создать Worker на Cloudflare

1. Зарегистрируйтесь на <https://dash.cloudflare.com> (бесплатно).
2. Слева выберите **Workers & Pages** → **Create application** → **Create Worker**.
3. Дайте имя (например, `sestram-reviews`) → **Deploy**.
4. Нажмите **Edit code**, удалите шаблон и вставьте содержимое файла
   [`worker.js`](./worker.js) → **Deploy**.

## Шаг 5. Добавить секреты

В настройках Worker: **Settings** → **Variables and Secrets** → **Add**:

| Имя           | Значение                                   | Тип            |
| ------------- | ------------------------------------------ | -------------- |
| `BOT_TOKEN`   | токен из шага 1                             | Secret         |
| `CHAT_ID`     | ID группы из шага 3 (например `-100...`)    | Text или Secret|
| `ALLOW_ORIGIN`| `https://urokisestram.com` (адрес сайта)   | Text           |

После добавления нажмите **Deploy**, чтобы изменения вступили в силу.

> `ALLOW_ORIGIN` можно не задавать (тогда будет `*` — принимать с любого сайта).
> Лучше указать адрес сайта, чтобы форму нельзя было дёргать с чужих страниц.

## Шаг 6. Подключить Worker к сайту

1. Скопируйте URL вашего Worker — он показан вверху страницы Worker, вид:
   `https://sestram-reviews.ВАШ-АккаунТ.workers.dev`.
2. Откройте `index.html`, найдите строку:

   ```js
   REVIEW_ENDPOINT = 'https://ЗАПОЛНИТЬ-ПОСЛЕ-ДЕПЛОЯ.workers.dev';
   ```

   и впишите туда свой адрес Worker.
3. Сохраните и закоммитьте изменение — готово.

(Либо просто пришлите мне URL Worker, и я подставлю его и закоммичу.)

## Проверка

Откройте сайт → раздел «Отзывы» → **Оставить отзыв** → напишите тестовый текст →
**Отправить**. В течение пары секунд сообщение должно прийти в Telegram-группу,
а на сайте появится экран «ДжазакиЛлаху хайран!».

## Стоимость

Бесплатного тарифа Cloudflare Workers (100 000 запросов в день) с огромным запасом
хватает для формы отзывов. Платить ничего не нужно.
