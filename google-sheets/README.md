# Заявки с сайта → Google-таблица (без формы)

Когда посетитель заполняет форму («получить гайды» / «получить книгу»), его данные
(**имя, почта, телефон, ник в Telegram + какой материал**) дописываются новой строкой
прямо в вашу **Google-таблицу**. Файлы при этом сразу скачиваются на сайте — независимо
от того, дошла заявка или нет.

Работает через **Google Apps Script** — маленький скрипт, привязанный к таблице.
Никакой Google Формы не нужно. Полностью бесплатно.

## Настройка (один раз, ~5 минут)

1. Создайте новую Google-таблицу на <https://sheets.google.com>.
2. В меню: **Расширения → Apps Script**.
3. Удалите весь код в редакторе и вставьте содержимое файла [`Code.gs`](./Code.gs).
   Нажмите **Сохранить** (значок дискеты).
4. Справа вверху: **Развернуть (Deploy) → Новое развёртывание (New deployment)**.
   - Тип: значок шестерёнки → **Веб-приложение (Web app)**.
   - **Execute as / Запуск от имени:** Я (ваш аккаунт).
   - **Who has access / Доступ:** Все (Anyone).
   - **Развернуть** → при первом разе Google попросит **авторизацию**: разрешите
     (через «Дополнительно → Перейти к проекту»).
5. Скопируйте **URL веб-приложения** (вид `https://script.google.com/macros/s/AKfyc.../exec`).
6. Пришлите этот URL мне — я подставлю его в `index.html` (`SHEET_ENDPOINT`) и закоммичу.

> Пока в `index.html` стоит `ВСТАВИТЬ_URL_WEB_APP`, отправка пропускается: на сайте всё
> работает и файлы выдаются, но заявки в таблицу не пишутся, пока URL не подключён.

## Как это работает

- Сайт отправляет `POST` на URL веб-приложения с полями `name, email, phone, tg, magnet`.
- `doPost` в `Code.gs` дописывает строку: `Дата | Материал | Имя | Почта | Телефон | Telegram`.
- Заголовки создаются автоматически при первой заявке.

### Без дублей

Если человек уже есть в таблице (совпадает **почта**, а при пустой почте — **телефон**),
новая строка не добавляется. Вместо этого обновляется дата и в колонку «Материал»
дописывается то, что он запросил. Пример: брал гайды, потом книгу → одна строка
с материалом «Гайды по планированию, Справочник постящейся».

## Если меняли код скрипта

После любого изменения `Code.gs` нужно заново **Развернуть → Управление развёртываниями →
изменить версию (New version)**, иначе старая версия продолжит работать по тому же URL.

## Проверка

Откройте сайт → «Гайды» → заполните форму → в таблице должна появиться новая строка
с контактами, а на сайте — кнопки скачивания.
