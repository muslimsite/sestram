# urokisestram.com

Сайт проекта «Сёстрам» — уроки и вебинары для сестёр от Солихи Умм Мусы (Sestram.com).

## Структура

- `index.html` — главная страница
- `Lessons.dc.html` — страница-предупреждение перед уроками по акыде (только для женщин)
- `support.js` — рантайм (необходим для работы страницы)
- `assets/` — изображения сайта

## Публикация через GitHub Pages

1. Залейте все файлы в репозиторий.
2. В репозитории: **Settings → Pages**.
3. В разделе **Build and deployment** → **Source** выберите **Deploy from a branch**.
4. Branch: `main`, папка: `/ (root)`. Сохраните.
5. Через минуту сайт будет доступен по адресу `https://ВАШ_ЛОГИН.github.io/НАЗВАНИЕ_РЕПО/`.
