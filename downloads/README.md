# Файлы для выдачи (лид-магниты)

Сюда кладутся файлы, которые посетитель скачивает после заполнения формы на сайте.
Хостятся прямо на GitHub Pages, ссылки на них формируются в `index.html`
(массивы `GUIDE_FILES` и `BOOK_FILE`).

> Файлы доступны по прямой ссылке всем, у кого она есть — для бесплатных
> материалов это нормально.

## Книга (готово ✅ — на Google Drive)

| Источник                          | Что                      |
| --------------------------------- | ------------------------ |
| Google Drive (прямая ссылка)      | Справочник постящейся     |

Книга не лежит в репозитории — она хостится на Google Drive, кнопка «Получить книгу»
ведёт на прямую ссылку скачивания (`BOOK_FILE.href` в `index.html`).
Чтобы заменить файл: загрузите новый PDF на Drive, откройте доступ «Все, у кого есть
ссылка», и подставьте новый ID в ссылку вида
`https://drive.google.com/uc?export=download&id=ВАШ_ID`.
