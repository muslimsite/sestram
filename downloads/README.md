## Куда попадают контакты (Google Форма)

Когда посетитель заполняет форму («получить гайды» / «получить книгу»), его данные
(имя, почта, телефон, ник в Telegram + какой материал) отправляются в **Google Форму**,
а оттуда автоматически собираются в Google-таблицу. Файлы при этом сразу скачиваются
на сайте — независимо от того, дошла заявка или нет.

### Как настроить (один раз)

1. Создайте Google Форму на <https://forms.google.com> со следующими вопросами
   (тип «Текст (строка)»), в любом порядке:
   - **Имя**
   - **Почта**
   - **Телефон**
   - **Ник в Telegram**
   - **Материал** (что скачали)
2. Пришлите ссылку на форму — я подставлю в `index.html` адрес отправки и ID полей.

> Технически: в `index.html` в объекте `GOOGLE_FORM` нужно указать `action`
> (вида `https://docs.google.com/forms/d/e/XXXX/formResponse`) и `entry.<id>`
> для каждого поля. Эти значения берутся из HTML страницы формы / из «пред­заполненной
> ссылки» (Get pre-filled link). Я могу извлечь их сам по ссылке на форму.

Пока `action` содержит `FORM_ID`, отправка пропускается — на сайте всё работает,
файлы выдаются, но контакты никуда не уходят, пока форма не подключена.
